== Description ==

The "Chikari Gallery Plugin" is a custom WordPress plugin that allows users to create and display image galleries on their websites. It provides a user-friendly interface to create gallery albums, add images to albums, and showcase the galleries using a shortcode. The plugin supports custom post types for gallery albums and utilizes WordPress media library for image management. It also includes a responsive slideshow feature with navigation controls and an option for automatic slide transitions. With its intuitive functionality, the plugin enables users to enhance their websites with visually appealing and interactive image galleries.

== Installation ==

This section describes how to install the plugin and get it working.

= AUTOMATIC INSTALLATION (EASIEST WAY) =

To do an automatic install of Chikari Gallery, log in to your WordPress dashboard, navigate to the Plugins menu and
click Add New.
In the search field type "Chikari Gallery". Once you have found it you can install it by simply
clicking "Install Now" and then "Activate".

= MANUAL INSTALLATION =

**Uploading in WordPress Dashboard**

* Download chikari-album-gallery.zip
* Navigate to the ‘Add New’ in the plugins dashboard
* Navigate to the ‘Upload’ area
* Select chikari-album-gallery.zip from your computer
* Click ‘Install Now’
* Activate the plugin in the Plugin dashboard

**Using FTP**

* Download chikari-album-gallery.zip
* Extract the chikari-album-gallery directory to your computer
* Upload the chikari-album-gallery directory to the /wp-content/plugins/ directory
* Activate the plugin in the Plugin dashboard
